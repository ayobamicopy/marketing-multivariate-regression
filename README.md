# Multi-Channel Marketing Optimization - Multiple Linear Regression

## Overview
This project builds a statistically robust Multiple Linear Regression layout using Python and `statsmodels`. It evaluates multicollinearity across channel combinations, analyzes p-values to prune non-significant variables, checks standard OLS structural assumptions, and provides actionable, data-backed advice for budget planning.
